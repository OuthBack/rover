## :scroll: Getting Started

To run the project it is just:

### :warning: Requirements

- Docker

### :gear: Installation

1. Choose your language
```sh
cd nodejs
```
or
```sh
cd ruby
```

2. Run docker build
```sh
docker build -t rover .
```

3. Run docker run
```sh
docker run rover
```

4. To change the input you just need to change the file "input.txt"
